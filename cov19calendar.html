週の陽性確認数(10万人あたり) vs 今週先週比¶

太い点線は、増減比(今週/先週)に変化がないときに、1週間どこまで増えるかを 表している。対策の効果が表れるのは 2 週間後なので、2週間後の予測を表す線 の方が適切かもしれないが、今週/先週比が一定という仮定がかなり乱暴なこと もあり、1週間先の方がチャートの使い勝手は良い気もする(よく分からない)。

def get_num_chg(dd_r, pp_r):
    """ 直近 1 週間分の 日付, 7日累積, 増減比(7日平均)
    
    増減比は、曜日毎の偏りを考慮する方式を採用した。
    (前の週の同じ曜日との比の平均)
    
    Parameters
    ----------
    dd_r, list of datetime : 日付のリスト
    pp_r, list of loat : 日ごとの陽性数のリスト
    
    Returns
    -------
    dd7, list of datetime : 日付のリスト(最終 7 日分)
    pp7, list of float : 7日累積
    cc7, list of float : 増減比(7日平均)
    
    Notes
    -----
    戻りは最終 7 日分に限定しているが、限定しない方が良いと思う。
    (計算できるものは全部返して、使う側で切り取ればよい)
    
    """
    pp_r = np.array(pp_r)
    cc = pp_r[7:]/pp_r[:-7] # 前の週の同じ曜日との比
    pp = pp_r[7:] # cc と長さを揃える
    n = len(pp)
    
    dd7 = dd_r[-7:]
    pp7 = cum(pp, 7)[-7:]
    cc7 = mav(cc, 7)[-7:]
    
    return dd7, pp7, cc7
